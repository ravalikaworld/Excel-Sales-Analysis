# Excel Sales Analysis Project

## Project Overview

This project analyzes a sales dataset using Microsoft Excel to identify business insights using data exploration and PivotTables.

## Dataset

* Source: Kaggle - Sample Sales Data

## Tools Used

* Microsoft Excel

## Skills Demonstrated

* Data Import
* Data Cleaning
* Filtering and Sorting
* Pivot Tables
* Business Analysis

## Key Insights

* Classic Cars appeared most frequently.
* Year 2005 had the highest sales/orders.
* Classic Cars generated the highest revenue (3,919,615).
* Vintage Cars generated revenue of 1,903,150.
* Revenue difference between Classic Cars and Vintage Cars: 2,016,465.
